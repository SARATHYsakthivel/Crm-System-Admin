# CRM System Admin - Automation Testing

A comprehensive automation testing framework for the CRM System using **Selenium WebDriver**, **Cucumber BDD**, and **Java**. This project focuses on testing the registration and login functionality of the CRM application.

## 🎯 Project Overview

This project automates the testing of critical user workflows in the CRM System:
- **User Registration** - Complete registration flow with validation
- **User Login** - Login functionality with valid credentials

## ✨ Features

- ✅ Automated user registration testing
- ✅ Automated login functionality testing
- ✅ BDD approach using Cucumber framework
- ✅ Selenium WebDriver for browser automation
- ✅ Chrome browser support
- ✅ JavaScript execution for element interaction
- ✅ Actions API for advanced user interactions
- ✅ TestNG integration for test execution
- ✅ Maven-based project management

## 📦 Prerequisites

Before you begin, ensure you have the following installed:

- **Java JDK 8+** ([Download](https://www.oracle.com/java/technologies/downloads/))
- **Maven 3.6+** ([Download](https://maven.apache.org/download.cgi))
- **Git** ([Download](https://git-scm.com/))
- **ChromeDriver** (compatible with your Chrome browser version)
  - Download from: [ChromeDriver Downloads](https://chromedriver.chromium.org/)
  - Or configure via WebDriverManager
